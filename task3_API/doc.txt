
google_sheets_api.py представляет скрипт, который получает координаты 
функцию get_location_by_address(api_key, address), которая использует Google Geocoding API 
для получения географических координат (широты и долготы) по заданному адресу. 
Затем, если информация была успешно получена, выводит координаты на экран.

get_location_by_address(api_key, address): Функция принимает два аргумента: 
api_key (ключ API Google Maps) и address (адрес, для которого требуется получить географические координаты).
Создается URL-запроса к Google Geocoding API с использованием адреса и ключа API.
Запрос отправляется с помощью библиотеки requests с использованием метода requests.get(url, params=params), 
где params - это словарь с параметрами запроса (адрес и ключ API).

Ответ от сервера получается в формате JSON и декодируется с помощью response.json().
Если ответ содержит статус "OK", это означает, что данные получены успешно.
Из ответа получается информация о местоположении из ключа 'results' и далее из 'geometry' и 'location'.
Затем извлекаются значения широты и долготы из данных о местоположении.
Возвращаются широта и долгота, если запрос был успешным. В противном случае, 
выводится сообщение об ошибке и возвращаются значения None, None.


В блоке if __name__ == '__main__': демонстрируется использование функции:

Указывается ключ API Google Maps в переменной api_key.
Задается адрес address (1600 Amphitheatre Parkway, Mountain View, CA).
Вызывается функция get_location_by_address(api_key, address) для получения координат.
Если получение координат прошло успешно (широта и долгота не равны None), они выводятся на экран. В противном случае выводится сообщение о неудаче.

Для работы с Google Sheets API необходимо установить библиотеку google-api-python-client
pip install google-api-python-client

credentials.json - это api_key

Для запуска скрипта необходимо выполнить команду:
python google_sheets_api.py
